This criterion evaluates a metric for each element, but then filters out the elements that have values within a min-max range. 

The `min` parameter sets the minimum value of the filter.

The `max` parameter sets the maximum value of the filter.

If the `clamp` parameter is true, the element values will be clamped to the range defined by min, max. 

The `data` parameter specifies the metric to evaluate the element values. Any of the other meshadaptor criteria.
